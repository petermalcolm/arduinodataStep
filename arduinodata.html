Now it's time to gather some data!
Follow these instructions to gather data from your Arduino device:
  1. Make sure the Arduino is plugged into one of the USB ports on your computer
  2. Click the Arduino (TM) Data Gatherer button below
  3. When a window pops up, choose (•) Open with [Java(TM) Web Start Launcher (default)] and click OK.
    4.       Java will start.
          A window will open that says Do you want to run this application?.
  4. Click the box next to "I accept the risk and want to run this application".
  5. Click Run.
    6.       Now you can gather data from the Arduino.
